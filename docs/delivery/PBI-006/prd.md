# PBI-006: Evidence-First Acceptance

## Status: Agreed

## Objective

Implement evidence-first acceptance format for all reviewer branches and completion contracts.

## Acceptance Criteria

- [ ] Reviewer branches (`code-reviewer`, `security-auditor`, `review`) require explicit evidence per AC
- [ ] Claim-to-evidence matrix format documented in `docs/evidence-first-acceptance.md`
- [ ] "Final Verification Checklist" in `docs/process-and-quality-gates.md` aligned with completion contracts
- [ ] No "done" without listing changes + evidence in COMPLETION_CONTRACT
