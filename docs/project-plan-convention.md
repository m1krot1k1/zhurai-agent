# Конвенция каталога `.plan/`

В корне репозитория приложения ведётся каталог **`.plan/`** — план, тудушки и заметки по направлениям с таймстемпами и отметками ✅.

Полная спецификация: [project-plan-dot-plan SKILL](../skills/project-plan-dot-plan/SKILL.md).

Кратко:

| Элемент | Назначение |
|---------|------------|
| `README.md` | Индекс, язык, часовой пояс таймстемпов |
| `todos.md` | Кратко: выполнено / очередь |
| `todos_full.md` | Подробный журнал прогресса |
| `napravlenie-*.md` | Один файл на домен/направление |

Язык записей — **русский по умолчанию** или язык чата с пользователем.

Параллельная работа оркестраторов и веток не освобождает от обновления `.plan/` после волн синтеза результатов.
